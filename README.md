# Flutter Web Portfolio

A beautiful, animated Flutter web portfolio for Sanchi Chauhan.

## Getting Started

1. Install Flutter (make sure you have Flutter web enabled)
2. Run `flutter pub get`
3. Run `flutter run -d chrome` or `flutter run -d web-server`
